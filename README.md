# Phân tích và So sánh các thuật toán sắp xếp 

## 1. Giới thiệu
Bài tập này thực hiện so sánh thời gian chạy thực tế của các thuật toán:
- Merge Sort
- Quick Sort
- Heap Sort
- C++ (std::sort)
- Numpy Sort

## 2. Cấu trúc thư mục
- `Heap_Sort.cpp` : Mã nguồn Heap sort viết dưới C++.
- `Merge_Sort.cpp` : Mã nguồn Merge sort viết dưới C++.
- `Quick_Sort.cpp` : Mã nguồn Quick sort viết dưới C++.
- `Std_Sort.cpp` : Mã nguồn sort của thư viện std.
- `Numpy_Sort.py` : Mã nguồn sort của thư viện Numpy.
- `Report.pdf`: Báo cáo chi tiết và biểu đồ.
- `test{n}.txt` : Là bộ dữ liệu đầu vào thứ n.

## Ghi chú về bộ dữ liệu
* **Test 1 -> Test 5:** Dữ liệu đầu vào là số thực.
* **Test 6 -> Test 10:** Dữ liệu đầu vào là số nguyên.
